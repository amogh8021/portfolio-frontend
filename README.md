# ♟️ Developer Portfolio

A modern chess-inspired developer portfolio built using React, TypeScript, Tailwind CSS, and Framer Motion.

The portfolio showcases my projects, technical skills, engineering journey, and provides a contact form powered by a Spring Boot backend.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio-frontend.git
cd portfolio-frontend
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file in the project root and add:

```env
VITE_API_URL=http://localhost:8080
```

> Replace the URL with your deployed backend URL when running in production.

### 4. Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

### 5. Build for production

```bash
npm run build
```

### 6. Preview the production build

```bash
npm run preview
```
